# Premium Bag Store

A full-stack e-commerce application for premium and luxury bags built using the MERN stack.

## 🚀 Features

### User Features
- User Registration & Login
- JWT Authentication
- Browse Products
- Product Details
- Shopping Cart
- Wishlist
- Order Placement
- Order History

### Admin Features
- Manage Products
- Manage Orders
- Manage Users
- Dashboard Analytics

## 🛠 Tech Stack

### Frontend
- React.js
- React Router
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
